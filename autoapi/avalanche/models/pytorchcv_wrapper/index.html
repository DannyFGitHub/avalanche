avalanche.models.pytorchcv_wrapper

This module provides acces to pytorchcv models. A general wrapper is available through get_model. For VGG, Resnet, DenseNet and Pyramidnet direct wrappers are provided.

Models pretrained on e.g. Imagenet don’t necessairly have the same structure as those used typically used for smaller datasets like Cifar10. So be carefull when adapting pretrained models for different datasets.

Not all options (e.g. growth rate for densenet, alpha in pyramidnet, bottlenecks…) are available through the direct wrappers. If a more specific models is required, it can be loaded through the general method.

Currently this module only wraps to pytorchcv models.
